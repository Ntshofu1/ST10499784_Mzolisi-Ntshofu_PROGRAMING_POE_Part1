# MyJavaChatApp

In this project, I built a simple Java chat application that allows a user to register and then log in.

I implemented input validation to make the system more secure and structured. 
For the username, I required it to include an underscore and be no longer than 5 characters. 
For the password, I added rules to ensure it is at least 8 characters long and includes a capital letter, a number, and a special character. I also validated the phone number so that it must start with +27 and not exceed 12 characters.

Once the user registers successfully, I created a login feature that checks whether the entered username and password match the stored details. If the login is correct, the user receives a welcome message; otherwise, they are prompted to try again.

This project helped me practice Java fundamentals such as methods, conditionals, loops, and user input using the Scanner class.
